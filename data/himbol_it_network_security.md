# Himbol Policy: Information Technology Network Security and Core Firewalls

**Document Reference: POL-IT-SEC07**
**Classification: Confidential - Internal Technical Use Only**
**Effective Date: January 1, 2026**

## 1. Perimeter Firewalls and Access Segregation

The core ledger infrastructure and M-Nakfa mobile application databases are hosted within a secure data room at the Asmara Head Office. Network connections from regional branches (Keren, Mendefera, Barentu, Adi Keyih, and Senafe) must tunnel exclusively through an encrypted Virtual Private Network (VPN) with multi-factor authentication. Direct public internet access from endpoint teller terminals is completely blocked by perimeter firewalls to prevent unauthorized data extractions.

## 2. Device Restrictions and Removable Media Banned

To defend our financial records against malware insertion and corporate espionage, all USB storage ports and removable optical drives are completely disabled at the system BIOS level across all Himbol workstations. Employees are strictly prohibited from connecting personal mobile hardware or consumer storage media to the company network. Unauthorized network attachment detections trigger an automated lock of the local machine and send an urgent alert to the IT Security Response Unit.
