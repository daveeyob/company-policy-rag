# Himbol Policy: Branch Cash Vault and Liquidity Logistics

**Document Reference: POL-OPS-V05**
**Effective Date: January 1, 2026**

## 1. Dual-Custody Access Rules

All physical vault installations at Himbol branch locations must be operated under strict dual-custody parameters. No single employee may possess both the combination code and the physical security key to a main branch vault. In regional branches like Senafe and Adi Keyih, the Combination Safe Code must be held exclusively by the Branch Manager, while the physical Dual-Lock Vault Key remains in the possession of the Senior Head Teller.

## 2. Daily Cash Holding Ceilings

To minimize operational loss risks from external threats, maximum overnight physical cash limits are designated as follows: Asmara Head Office: 5,000,000 ERN; Keren and Mendefera Main Branches: 1,500,000 ERN; Barentu, Adi Keyih, and Senafe Regional Branches: 750,000 ERN. Any physical cash volumes exceeding these specific ceilings must be securely escorted to the designated central banking depository via approved armed transport vehicles.
